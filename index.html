Problem / Pain

ขี้เกียจรอการซื้อของแบรนด์เนมหรือการต่อคิวด้วยตนเองนาน ๆ

User Interviews

เลือกทำ User Interview กับ Users เพื่อทราบถึงข้อมูลและความต้องการจริงๆ ของ Users เพื่อที่จะนำข้อมูลมาวิเคราะห์ และปรับกลยุทธ์ในการออกแบบแอพให้แม่นยำและตอบโจทย์ความต้องการของ Users ให้ได้มากที่สุด โดยการสัมภาษณ์จะมีการเตรียมคำถามไปเป็นข้อ แต่จะเป็นการสัมภาษณ์แบบพูดคุยเพื่อที่จะให้ User ได้เล่าเรื่องพูดคุยได้อย่างเป็นธรมชาติ และจะมีคำถามที่ได้เตรียมไว้ถามในจุดที่สำคัญและจุดที่ต้องหารเน้นย้ำ



คำถามที่ใช้ในการพูดคุยและสัมภาษณ์

ลูกค้าซื้อของแบรนด์เนมบ่อยมากแค่ไหน?

ลูกค้ามีวิธีการซื้อของที่อยากได้ในตอนที่อยู่ห่างไกลจากร้านค้าที่ขายสินค้าอย่างไร ?

ลูกค้ารู้สึกขี้เกียจไหมที่ต้องหาเวลาไปซื้อของที่อยากได้ในเวลาที่จำกัดเวลาการขายสินค้าสินค้าชิ้นนั้น ?

ลูกค้าเคยไม่มีเวลาไปซื้อของด้วยตัวเองหรือไม่ ? (ถ้ามี) : ลูกค้าสามารถหาคนฝากซื้อที่ไว้วางใจได้หรือไม่ ?



สรุปคำตอบที่เป็นข้อมูลสำคัญ
